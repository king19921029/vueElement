# YQ-admin-web
vue后台管理系统，借用了macrozheng大佬的mall-admin-web的项目，在这基础上开发

## 项目介绍
主要包括 服务管理、订单管理、运营管理、内容管理、权限管理

### 项目演示
macrozheng大佬的mall-admin-web演示地址：[http://39.98.190.128/index.html](http://39.98.190.128/index.html)
